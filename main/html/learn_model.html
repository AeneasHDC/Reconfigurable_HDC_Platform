AeneasHDC FAQ:

What is AeneasHDC? AeneasHDC is the first automatic open-source framework in the literature to facilitate the generation of flexible hardware accelerators for hyperdimensional computing (HDC) learning problems. The environment is extremely customizable, supporting a wide range of the most common techniques adopted in the literature for learning models, enabling users to easily assess the impact of different design choices on model accuracy, memory usage, execution time, energy consumption, and area requirements.

What is the purpose of AeneasHDC? The main goal of AeneasHDC is to provide a complete and automatic environment to generate flexiblehardware accelerators for HDC learning problems, allowing the user to easily explor the impact of different design choices on model performance as well as hardware requirements.

How to use AeneasHDC? The intuitive AeneasHDC's GUI allows users to define the specifics of the problem, such as dataset characteristics, the desired programming language, the target FPGA device and the model details. It also allows to distribute tasks between software and hardware implementations, allowing for flexible customization of the hardware accelerator. The desired model is automatically instantiated and evaluated both in software and hardware and all the report data are made available in PDF. From the home page, click on the "Configure Your Model" button to open the configuration page, and adjust your model's parameters.

Can I use the environment without GUI? Sure, all the software and hardware libraries of AeneasHDC are open-source and completely free. You can just import the required class and use the model how u prefer. You can find all the infos in the AeneasHDC PDF manual.
